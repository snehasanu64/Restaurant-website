# 🍛 Donne Biryani House
 
A responsive front-end website for a local restaurant in Jayanagar, Bangalore.
Built to bring the restaurant online with a clean menu and a heritage-style look.
 
## ✨ Features
 
- Home page with restaurant story and legacy
- Full menu browsing by category
- Online order section
- Customer reviews section
- Fully responsive — works on mobile and desktop
