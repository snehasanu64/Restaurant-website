# 🍛 Donne Biryani House - Premium Restaurant Website

A modern, high-end restaurant website built with a **Royal Heritage** theme. This project features a sophisticated multi-page architecture, dynamic cart state management, and a premium UI/UX design tailored for a legendary Bangalore biryani brand.

![Design](https://img.shields.io/badge/Theme-Royal%20Heritage-maroon)
![Architecture](https://img.shields.io/badge/Structure-Multi%20Page-gold)

## ✨ Key Features

*   **🏆 Premium Design System**: A bespoke "Royal Heritage" aesthetic featuring a warm cream background, deep maroon accents, and elegant gold details.
*   **📂 Multi-Page Architecture**: Structured for professional maintenance with dedicated pages for Home, Menu, and Checkout.
*   **🛒 Smart Add-to-Cart System**: 
    *   Interactive menu cards with dynamic state transitions.
    *   Initially shows a high-impact "Add to Cart" button that transforms into `[-] 1 [+]` quantity controls upon click.
*   **🔄 Persistent Cart Logic**: Cart items are synced across all pages using local storage and a centralized JavaScript state engine.
*   **📱 Fully Responsive**: Optimized for seamless experiences on desktop, tablet, and mobile devices.
*   **⚡ Modern Typography**: Uses premium Google Fonts (*Playfair Display* for headings and *Plus Jakarta Sans* for body).

## 🛠️ Built With

*   **HTML5**: Semantic structure for SEO and accessibility.
*   **CSS3**: Vanilla CSS with custom properties (variables), glassmorphism effects, and premium animations.
*   **JavaScript (ES6)**: Modular logic for cart state management and dynamic UI updates.
*   **Google Fonts**: High-impact typography.
